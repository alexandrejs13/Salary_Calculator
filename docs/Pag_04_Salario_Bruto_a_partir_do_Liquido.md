  Página 04 — Cálculo do salário bruto a partir do salário líquido
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

OBJETIVO
Calcular o salário bruto necessário para que o funcionário receba um valor líquido informado, considerando todas as incidências e descontos do país selecionado.

  País ▢                     Frequência anual (fixo) ▢ [auto]

  Salário líquido desejado ▢


────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
  2. Cálculo inverso — Como o sistema encontra o salário bruto
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
  O sistema encontra o salário bruto necessário resolvendo a equação:

      bruto - descontos_do_empregado(bruto) = líquido_informado

  Onde “descontos_do_empregado(bruto)” inclui:
    • INSS / Previdência obrigatória
    • Imposto de renda (IR / IRRF)
    • Previdência complementar (se houver)
    • Contribuições específicas do país
    • Outras deduções configuradas por país

  O cálculo é feito por iteração numérica até que o valor líquido seja alcançado com precisão.
